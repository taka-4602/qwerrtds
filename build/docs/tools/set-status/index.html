Set Statusの使い方

概要​

Set Statusは、Discordアカウントのプレゼンス状態（ステータス）を自由に設定できるツールです。通常のDiscordクライアントでは設定できない詳細なカスタムステータスや、Spotify画像を使用した特殊なステータス表示が可能です。

基本設定​

トークン設定​

MTM1ODc4MTE3NTIzMzI1MzU*****
  • 使用するDiscordユーザートークンを入力
  • ボットトークンではなくユーザートークンを使用

アクティビティ設定​

アクティビティタイプ​

利用可能なアクティビティタイプ:

  • プレイ中 (Playing): ゲームをプレイしている状態
  • 配信中 (Streaming): ライブ配信している状態
  • 音楽再生中 (Listening): 音楽を聴いている状態
  • 視聴中 (Watching): 動画を視聴している状態

アクティビティ詳細​

  • アクティビティ名:
    • 例: VSCodium
    • メインのアクティビティ名を設定
  • 詳細:
    • 例: unko.pyを編集中
    • アクティビティの詳細情報
  • 状態:
    • 例: Workspace: UnkoProject
    • 現在の状況や場所

カスタム画像設定​

大きい画像​

  • Spotify URL: SpotifyのトラックまたはアルバムURL
    • Spotifyの画像のみステータスに設定可能
    • 例: https://open.spotify.com/album/xxxxxx
  • 大きい画像のテキスト:
    • 画像にカーソルを合わせたときに表示されるツールチップ

小さい画像​

  • Spotify URL: 小さいアイコン用のSpotifyリンク
    • 例: https://open.spotify.com/track/5pZVsZ8TOGly1KnYFmZ61B
  • 小さい画像のテキスト:
    • 小さい画像のツールチップテキスト

実行手順​

  1. トークン入力: 有効なDiscordユーザートークンを設定
  2. アクティビティタイプ選択: プレイ中、配信中、音楽再生中、視聴中から選択
  3. 詳細情報入力: アクティビティ名、詳細、状態を設定
  4. 画像設定: Spotify URLを使用して大きい画像と小さい画像を設定
  5. ステータス設定: 設定内容を適用
  6. 切断: ステータス設定を解除してオフライン状態に変更

制御オプション​

ステータス管理​

  • ステータス設定: 入力した内容でプレゼンス状態を更新
  • 切断: 現在のステータスを解除してオフライン表示

荒らし活用例​

偽装ステータス​

  1. 開発者偽装: VSCodeでコーディング中に偽装
  2. 配信者偽装: 実際は配信していないのに配信中ステータス
  3. 音楽偽装: 聴いていない楽曲の再生ステータス表示

混乱誘発​

  • サーバーメンバーを困惑させる不自然なステータス
  • 運営陣の偽装ステータスでサーバー内混乱
  • 一斉に同じ偽装ステータスで集団行動演出

設定例​

開発者偽装​

アクティビティタイプ: プレイ中
アクティビティ名: Visual Studio Code
詳細: malware.pyを編集中
状態: Workspace: HackProject

配信者偽装​

アクティビティタイプ: 配信中
アクティビティ名: Twitch配信
詳細: 荒らし実況中
状態: 視聴者数: 1000人

音楽偽装​

アクティビティタイプ: 音楽再生中
アクティビティ名: Spotify
詳細: 人気楽曲
大きい画像: https://open.spotify.com/track/xxxxx
