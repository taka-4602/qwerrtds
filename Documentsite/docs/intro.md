# OZEU ドキュメントサイトへようこそ

このサイトでは、各種 Discord ツールの使用方法を詳しく説明します。

## クイックスタート

### 1. 基本準備

まずは以下の基本設定を行います：

- **[トークン取得](./getting-started/token-guide)**：Discord トークンの取得方法  
- **[開発者モード](./getting-started/developer-mode)**：Discord 開発者モードの有効化  
- **[用語集](./getting-started/glossary)**：専門用語の理解

### 2. ツールの使用

準備が完了したら、以下のツールを利用できます。

#### スパム系ツール

> ※以下のツール名はドキュメント上の項目です。実際の悪用は厳に慎んでください。

:::tip 便利なリンク
全ツールを一覧で確認: **[OZEU ツール一覧サイト](https://tools.ozeu.net/)**
:::

- **[DM Spam](./tools/dm-spam)**：プライベートメッセージの一括送信  
- **[Group Spam](./tools/group-spam)**：グループチャットでのメッセージ送信  
- **[Raid](./tools/raid)**：サーバー内での大規模メッセージ送信  
- **[Reaction Spam](./tools/reaction-spam)**：メッセージへの大量リアクション  
- **[Webhook Spam](./tools/webhook-spam)**：Webhook を使用したメッセージ送信

#### ユーティリティツール

- **[Joiner](./tools/joiner)**：サーバーへの一括参加  
- **[Token Checker](./tools/token-checker)**：トークンの有効性確認（[トークンガイド](./getting-started/token-guide) を参照）  
- **[Set Status](./tools/set-status)**：カスタムステータス設定  
- **[Slash Command Auto](./tools/slash-command-auto)**：スラッシュコマンド自動実行

---

## はじめての方へ

1. **[トークン取得方法](./getting-started/token-guide)** から始めてください。  
2. **[用語集](./getting-started/glossary)** で専門用語を確認してください。  
3. 使用したいツールのページで詳細な手順を確認してください。  
4. **[開発者モード](./getting-started/developer-mode)** を有効化して ID を取得してください。

---

## よく使われる組み合わせ

> 下記はドキュメント内でよく参照される「組み合わせ例」です。実際の運用ではサービス規約・法律を順守してください。

### 基本的な荒らしセット（教育目的の例示）

1. [Token Checker](./tools/token-checker) でトークンの有効性確認  
2. [Joiner](./tools/joiner) でターゲットサーバーに参加  
3. [DM Spam](./tools/dm-spam) または [Raid](./tools/raid) で実行

### 高度な荒らしセット（教育目的の例示）

1. [Set Status](./tools/set-status) で偽装ステータス設定  
2. [Webhook Spam](./tools/webhook-spam) でレート制限回避（※技術的説明のための記載であり悪用禁止）  
3. [Reaction Spam](./tools/reaction-spam) で追加の影響を与える

---

## 注意事項

- 各ツールの使用は **自己責任** で行ってください。  
- [トークン](./getting-started/token-guide) は重要な認証情報です。適切に管理してください。  
- 初回使用時は **[用語集](./getting-started/glossary)** で用語を確認してください。  
- 本ドキュメントは教育・検証目的の説明を意図しており、違法・悪意のある活動を助長する目的での利用は固く禁じます。
