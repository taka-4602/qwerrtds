# Joinerの使い方

:::info ツール一覧
すべてのツールをまとめて確認: **[OZEU ツール一覧サイト](https://tools.ozeu.net/)**
:::

**ツールURL**: https://lookjbn.github.io/ozeu-web-raider

## 概要
Joinerとは、大量のトークンを使用してアカウントを特定のサーバーに一斉参加させるツールです。参加後は他のツールと組み合わせて荒らしを実行したり、自サーバーをブーストさせたりできます。また、同時に大量の通報を行いサーバーを削除させることも可能です。

## 基本的な使い方

![joiner](スクリーンショット 2025-11-12 19.37.16.png)

### 手順
1. **トークンを入力**：使用するトークンを設定します
2. **招待コードを入力**：参加したいサーバーの招待コードを入力します
3. **Join開始を押す**：実行すると、完了したトークンと失敗したトークンが表示されます
4. **結果を他ツールで活用**：成功したトークンをコピーし使って他のツールで荒らしを実行します

## アドバンスオプション

アドバンスオプションをクリックすると詳細設定が可能です：

![アドバンス設定](スクリーンショット 2025-11-12 19.43.01.png)

### 設定項目
- **サーバーニックネーム**：参加時に設定するニックネーム
- **プロフィールメッセージ**：プロフィールに設定するメッセージ
- **絵文字リアクション用メッセージURL**：DiscordのメッセージURLを設定すると、そのメッセージに大量の絵文字リアクションを実行
- **2captcha APIキー**：設定するとCAPTCHA認証を自動解決
