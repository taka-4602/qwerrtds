# DM Spamの使い方

## 概要
DM Spamは、[トークン](../getting-started/token-guide.md)を使用して複数のユーザーに同時にプライベートメッセージを送信するツールです。効率的なメッセージ配信により、大規模な通知や連絡が可能です。

## 事前準備

### 必要な情報
- **[トークン](../getting-started/token-guide.md)**: Discordユーザートークンが必要
- **[ユーザーID](../getting-started/developer-mode.md)**: 送信対象のユーザーIDリスト
- **メッセージ内容**: 送信するメッセージテキスト

### 準備手順
1. [トークンを取得](../getting-started/token-guide.md)してください
2. [開発者モードを有効化](../getting-started/developer-mode.md)してユーザーIDを取得
3. 送信したいメッセージ内容を準備

## 基本設定

### トークン設定
```
トークン
MTM1ODc4MTE3NTIzMzI1MzU*****
```
- [取得したトークン](../getting-started/token-guide.md)を入力
