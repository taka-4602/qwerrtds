# Slash Command Autoの使い方

## 概要
Slash Command Autoは、Discordサーバー内のスラッシュコマンドを自動検索・取得し、大量実行できるツールです。ボットが提供するスラッシュコマンドを悪用してサーバーに負荷をかけたり、コマンドスパムを実行したりできます。

## 基本設定

### 1. Discord設定
- **Discord Token**: 使用するDiscordユーザートークンを入力
- **サーバーID**: ターゲットとなるサーバーのID

### 2. コマンド検索設定
- **検索するコマンド名**: 検索したいコマンド名を入力（部分一致対応）
- **コマンドを取得**: ボタンをクリックしてサーバー内のスラッシュコマンド一覧を取得

## 実行手順

### 基本操作
1. **トークン設定**: 有効なDiscordユーザートークンを入力
2. **サーバー指定**: 荒らし対象のサーバーIDを設定
3. **コマンド検索**: 利用したいコマンド名を部分検索で入力
4. **コマンド取得**: 「コマンドを取得」ボタンでサーバー内のスラッシュコマンド一覧を表示
5. **コマンド選択**: 取得されたコマンドから実行したいものを選択
6. **自動実行**: 選択したコマンドを大量自動実行

### 管理機能
- **ログ**: 実行状況とエラーログを表示
- **UI全リセット**: 設定内容をすべてクリアして初期状態に戻す

## 荒らし戦術

### コマンドスパム
1. **音楽ボットコマンド**: `/play`, `/volume`, `/skip`などを大量実行
2. **管理系コマンド**: `/kick`, `/ban`, `/mute`などの権限コマンド試行
3. **情報系コマンド**: `/help`, `/info`などで大量の応答メッセージ生成

### ボット負荷攻撃
- 処理の重いコマンドを同時大量実行
- データベースアクセスが必要なコマンドでサーバー負荷増大
- API制限到達によるボット機能停止

### チャンネル荒らし
- 応答メッセージが大量生成されるコマンドでチャンネルを埋める
- エラーメッセージを意図的に発生させて混乱誘発
- 重要な会話を大量の自動応答で流す

## 効果的な使用例

### 音楽ボット攻撃
```
検索コマンド名: play
取得されるコマンド: /play, /playlist, /playback
実行効果: 音楽再生の混乱、キュー荒らし
```

### 管理ボット攻撃
```
検索コマンド名: mod
取得されるコマンド: /moderation, /moderate, /mod
実行効果: 権限エラーメッセージ大量生成
```

### ユーティリティボット攻撃
```
検索コマンド名: info
取得されるコマンド: /info, /userinfo, /serverinfo
実行効果: 情報表示メッセージでチャンネル埋め
```

## 注意点

### 実行前確認
- コマンドの権限要件を確認
- 実行回数とタイミングを調整
- エラーログを監視して問題を早期発見

### 効率的な荒らし
- 部分一致検索で関連コマンドを一括取得
- 複数のボットコマンドを同時攻撃
- ログ機能で実行状況を把握しながら調整
