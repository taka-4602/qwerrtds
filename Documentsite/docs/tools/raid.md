# Raidの使い方

:::info ツール一覧
すべてのツールをまとめて確認: **[OZEU ツール一覧サイト](https://tools.ozeu.net/)**
:::

## 概要
Raidは複数のトークンを使用してDiscordサーバーで大量のメッセージ送信を行うツールです。メンション機能や投票機能、メッセージ転送機能を組み合わせた包括的なスパムツールです。
また▶マークのあるところはタップまたはクリックすることによって開きます

**ツールURL**: https://lookjbn.github.io/ozeu-web-raider/

![スクショ](/img/tools/image-1.png)
## 基本設定

### トークン設定
```
カンマ / 改行 / スペースで区切る
```
- 複数のトークンを入力
- 形式: `token1, token2, token3` または改行区切り

### ターゲット設定
- **サーバーID**: 対象となるDiscordサーバーのID
- **チャンネルID**: メッセージを送信するチャンネルのID
- **チャンネル自動取得**: チェックすると利用可能なチャンネルを自動検出

## メッセージ設定

### 基本メッセージ
- **メッセージ内容**: 送信するテキストを入力
- **送信間隔**: メッセージ送信の間隔を秒単位で設定（0秒 = 最速）
- **送信上限**: 送信するメッセージの最大数（0 = 無制限）

## メンション設定

### 手動メンション
- **メンションユーザーID**: メンションする特定のユーザーIDをカンマ区切りで入力

### 自動メンション
- **メンション取得用チャンネルID**: ユーザーリストを取得するチャンネル
- **取得する人数**: メンションするユーザー数を指定
- **総数取得**: チェックするとチャンネル内全ユーザーを取得
- **ランダムメンション**: 指定した人数をランダムでメンション

## 投票設定

### 投票スパム
- **投票タイトル**: 投票の件名
- **選択肢**: カンマ区切りで選択肢を入力
  - 例: `選択肢1,選択肢2,選択肢3`

## メッセージ転送設定

### 転送機能
- **転送するメッセージURL**: 他のチャンネルのメッセージURLを入力
- そのメッセージ内容を大量転送

## 実行手順

1. **トークン入力**: 使用するトークンを設定
2. **ターゲット設定**: サーバーIDとチャンネルIDを入力
3. **メッセージ設定**: 送信内容と間隔を設定
4. **オプション設定**: メンション、投票、転送を必要に応じて設定
5. **実行開始**: 「実行」ボタンをクリック

## 制御オプション

### 実行中の操作
- **スパム停止**: 実行中のスパムを緊急停止
- **サーバー退出**: 使用中のトークンでサーバーから一括退出

## 効果的な使い方

### 基本戦術
1. **大量メッセージ**: 送信間隔0秒で高速スパム
2. **メンション爆撃**: 全ユーザーをランダムメンションで混乱誘発
3. **投票スパム**: 大量の投票で通知を氾濫させる
4. **転送攻撃**: 重要メッセージを大量転送で埋没させる

### 組み合わせ戦術
- メッセージ + メンション: 最大限の通知騒音
- 投票 + 転送: チャンネルの機能麻痺
- 複数チャンネル同時攻撃: サーバー全体への影響拡大
