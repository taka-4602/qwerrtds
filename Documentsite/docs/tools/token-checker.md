# Token Checkerの使い方

:::info ツール一覧
すべてのツールをまとめて確認: **[OZEU ツール一覧サイト](https://tools.ozeu.net/)**
:::

## 概要
Token Checkerは、Discordトークンの有効性を一括で確認できるツールです。大量のトークンを効率的に検証し、有効なもののみを抽出して他のツールで活用できます。無効化されたトークンを除外することで、荒らし作業の効率を大幅に向上させます。

## UI構成と使い方

### トークン入力エリア
```
トークン
カンマ／改行／空白で区切る
```
- **入力形式**: 複数のトークンを以下の形式で入力
  - **改行区切り**: 1行に1つのトークン
  - **カンマ区切り**: `token1,token2,token3`
  - **空白区切り**: スペースで区切って入力

### 実行ボタン
- **チェック**: 入力されたトークンの検証を開始

### ログ表示エリア
```
ログ
ここに処理ログが表示されます
```
- 検証の進行状況をリアルタイムで表示
- エラーメッセージや警告を確認可能
- 処理完了時間とサマリー情報

### 結果表示エリア

#### 有効トークン
```
有効
 有効をコピー
```
- 検証に成功したトークンを表示
- 「有効をコピー」ボタンで一括コピー可能

#### ロックトークン  
```
ロック
 ロックをコピー
```
- アカウントがロックされているトークンを表示
- 一時的に使用不可能だが将来的に回復の可能性

#### 無効トークン
```
無効
 無効をコピー
```
- 完全に無効化されたトークンを表示
- 削除対象として管理

## 実行手順

### 基本操作
1. **トークン入力**: テキストエリアに検証したいトークンを入力
2. **検証開始**: 「チェック」ボタンをクリック
3. **ログ監視**: ログエリアで進行状況を確認
4. **結果確認**: 有効・ロック・無効の分類結果を確認
5. **トークン抽出**: 必要な分類のトークンをコピーボタンで取得

### 結果の活用
- **有効トークン**: 荒らしツールに即座に投入可能
- **ロックトークン**: 回復待ちまたは軽微な作業用
- **無効トークン**: リストから除外して管理

## 検証結果の詳細

### 有効トークンの特徴
- API呼び出しが正常に応答
- ユーザー情報の取得が可能
- 各種Discord機能にアクセス可能

### ロックトークンの特徴
- アカウントが一時的に制限状態
- 電話番号認証等で回復の可能性
- 限定的な機能のみ利用可能

### 無効トークンの特徴
- 認証エラー（401 Unauthorized）
- アカウント削除または永久停止
- 完全に使用不可能

## 荒らし運用での効率的活用

### 事前準備フロー
1. **大量トークン収集**: 各種方法で取得したトークンを準備
2. **Token Checkerで一括検証**: 全トークンの状態を確認
3. **分類別管理**: 有効・ロック・無効に分けて保存
4. **有効トークン投入**: DM Spam、Raid、Joiner等に活用

### 定期メンテナンス
- **週次検証**: 保有トークンの定期的な状態確認
- **新規トークン検証**: 新しく取得したトークンの即座チェック
- **ロックトークン再検証**: 一定期間後の回復状況確認

## 効率化のメリット

### 実行時間短縮
- 無効トークンによる実行失敗を事前排除
- 有効トークンのみで確実な荒らし実行
- エラーハンドリング時間の削減

### リソース最適化
- 無駄な API 呼び出しを削減
- サーバー負荷の軽減
- ツール実行の成功率向上

## 注意点

### 検証頻度の調整
- 過度な検証はDiscord API制限に抵触
- 適切な間隔での実行を推奨
- レート制限を考慮した運用

### セキュリティ管理
- トークン情報の適切な取り扱い
- ログファイルの安全な管理
- 第三者への情報流出防止
