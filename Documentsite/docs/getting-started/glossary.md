# 用語集

このページでは、サイト内で使用される専門用語を説明します。

## あ行

### API
Application Programming Interfaceの略。アプリケーション間でデータをやり取りするためのインターフェース。

### 荒らし
セキュリティテストやスパム送信を目的とした一連の行動。本サイトでは実験的・教育的な文脈で使用。

### 開発者モード
Discordの設定で有効化できる機能。[ユーザーID](#ユーザーid)や[サーバーID](#サーバーid)などの取得が可能になります。
→ 詳細: [開発者モード有効化方法](./developer-mode.md)

## さ行

### サーバーID
Discordサーバーの一意識別番号。[開発者モード](./developer-mode.md)を有効化して取得します。

### スパム
大量のメッセージや通知を送信する行為。[DM Spam](../tools/dm-spam.md)や[Raid](../tools/raid.md)などのツールで実行。

### スラッシュコマンド
Discordで「/」から始まるコマンド機能。[Slash Command Auto](../tools/slash-command-auto.md)で自動実行可能。

## た行

### トークン
Discordアカウントの認証に使用される文字列。各種ツールの動作に必要な重要な情報。
TOKENを使うと各種ツールにアカウントを動かす権限を渡せるログインして使おう
→ 詳細: [トークン取得方法](./token-guide.md)

### チャンネルID
Discordチャンネルの一意識別番号。[開発者モード](./developer-mode.md)で取得可能。

## は行

### Webhook
外部からDiscordチャンネルにメッセージを送信するための機能。[Webhook Spam](../tools/webhook-spam.md)で活用。

## ら行

### レート制限
APIの呼び出し頻度に対する制限。過度なリクエストを防ぐためのシステム。

### リアクション
Discordメッセージに付与できる絵文字反応。[Reaction Spam](../tools/reaction-spam.md)で大量実行可能。

## や行

### ユーザーID
Discordユーザーの一意識別番号。[開発者モード](./developer-mode.md)を有効化して取得します。
