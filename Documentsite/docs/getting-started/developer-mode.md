# 開発者モードの有効化

DiscordでサーバーIDやチャンネルIDを取得するには、開発者モードを有効にする必要があります。

## 設定手順

1. Discordの設定を開きます
2. 「詳細設定」または「Advanced」を選択します
3. 「開発者モード」をオンにします

## IDの取得方法

### サーバーIDの取得
- サーバー名を右クリック
- 「IDをコピー」を選択

### チャンネルIDの取得
- チャンネル名を右クリック
- 「IDをコピー」を選択

### ユーザーIDの取得
- ユーザー名を右クリック
- 「IDをコピー」を選択
